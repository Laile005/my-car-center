# マーケティング改善ロードマップ

## 基本方針

広告費をかけずに伸ばす。

優先順位は次の通り。

1. 中古車問い合わせ
2. 板金塗装・保険修理問い合わせ
3. 採用問い合わせ
4. 車検・日常整備
5. 新車相談

## 短期でやること

### Search Consoleの確認

Search Consoleのデータが溜まり始めたら、週1回見る。

特に見るもの:

- 中古車関連クエリ
- 板金塗装関連クエリ
- 保険修理関連クエリ
- 整備士求人関連クエリ
- 表示回数はあるがCTRが低いページ
- 8位から20位あたりの記事

対応:

- CTRが低いページはtitleとdescriptionを改善
- 10位前後の記事は本文追記と内部リンク追加
- 表示されていない重要ページは見出しや本文の不足を確認

### GA4の確認

見るイベント:

- `phone_click`
- `goo_net_click`
- `used_car_stock_view`
- `article_card_click`
- `recruit_form_submit_success`

見たい判断:

- 中古車ページ閲覧から電話につながっているか
- Goo-netへ流れすぎて自社サイトに戻ってこない問題がないか
- 記事からサービスページへ移動しているか
- 採用ページでフォーム完了まで進んでいるか

### Clarityの確認

見るもの:

- デッドクリック
- スマホで押されているが反応しない場所
- 電話ボタン付近の迷い
- 中古車カードのクリック
- コラムの読了率

## 中古車強化

中古車は問い合わせが伸びているため、継続して強化する。

追加したい記事:

- 修復歴ありの中古車は大丈夫？
- 修理前写真がある中古車を選ぶメリット
- 中古車を条件から探してもらうメリット
- 中古車は販売店と整備工場どちらに相談する？
- 支払総額・車両本体価格・諸費用の見方
- 中古車購入後にかかる整備費用
- 安い中古車で確認したい修理歴と消耗品
- 軽自動車の中古車を探す時のチェックポイント
- 通勤用の中古車を予算内で探す時の考え方
- 家族用の中古車を選ぶ時に見るポイント

ページ改善候補:

- 仕入れ時写真・修理前写真を掲載できる枠を作る
- 実際に販売した車の事例を追加
- 「条件から探す相談」をファーストビューまたは直下でさらに強める
- Goo-netへの遷移前に電話相談導線を置く

## 板金塗装・保険修理強化

追加したい記事:

- 保険修理で代車は借りられる？
- 修理中に車を預ける流れ
- 修理後に車を届けてもらえる？
- 保険会社とのやり取りでお客様がやること
- へこみ修理は交換になることがある？
- バンパー修理と交換の判断
- 色合わせが難しい車の修理で確認したいこと
- 小さなキズを放置しない方がよい理由
- 修理見積りで見るべき項目
- 保険を使うか自費で直すか迷った時の考え方

ページ改善候補:

- 施工写真が増えたら施工事例を独立強化
- 代車の写真を載せる
- 塗装設備の写真を載せる
- 保険修理の流れを図解化する

## 採用強化

追加したい記事:

- 整備士求人は年収だけで選んでいい？
- 整備士で残業が少ない職場を探すには？
- 未経験から整備士を目指せる？
- 板金塗装は未経験から始められる？
- 職場見学で聞くべきこと
- 資格取得支援で確認すべきこと
- 大手整備工場と地域密着の整備工場の違い
- 日曜・祝日休みの整備士求人を探す時のポイント
- 整備士経験者が転職で重視したい働き方

ページ改善候補:

- 「カジュアル面談」導線を強める
- 1日の流れを写真つきで出す
- 実際の作業場写真を追加
- 未経験者向け、経験者向けの導線を分ける

## 月次レポートの型

毎月見る項目:

- Search Consoleの検索クエリ上位10件
- 表示回数が増えたページ
- CTRが低いページ
- GA4の電話クリック数
- GA4のGoo-netクリック数
- 中古車ページの閲覧数
- 採用ページの閲覧数とフォーム成功数
- Clarityで目立つデッドクリック

改善アクション:

- title/description修正
- CTA位置変更
- 記事追記
- 内部リンク追加
- 画像追加
- 電話導線改善

## 判断基準

- 中古車ページが見られているなら、中古車記事と事例を増やす。
- Goo-netクリックが多いなら、自社サイト内で相談する理由をさらに強める。
- 電話クリックが少ないなら、電話導線の位置と文言を見直す。
- 記事は読まれるが問い合わせがないなら、記事下部のCTAを強化する。
- 採用ページが読まれているが応募がないなら、職場見学・カジュアル面談をさらに軽く見せる。
