# サイト共通ガイド

新しいページ、記事、導線を追加するときに最初に確認する、山本マイカーセンターの共通ルールです。

## このサイトが伝える約束

> 新車・中古車の購入から、納車後の整備・修理まで。一貫して対応します。

山本マイカーセンターは、整備・修理を本業とする地域密着の整備工場です。車を売ることだけ、修理することだけではなく、使い方や状態を踏まえて、購入前から納車後まで相談できることが大きな特長です。

ページごとに主役は変えてよいですが、この約束と矛盾する表現は使いません。

## 優先する事業と導線

1. 板金塗装・保険修理
2. 中古車の在庫・条件からの車探し
3. 車検・点検・日常メンテナンス
4. 採用
5. 新車相談

- 中古車は、掲載在庫を見る導線と「希望条件から探す」相談を並べる。後者を弱くしない。
- 修理は、車のお預かり、代車、必要に応じたお届け、保険会社とのやり取りの補助など、実際に提供する安心を示す。
- 新車は国内全メーカーの相談に対応するが、試乗やメーカー保証の確認はディーラーの良さも正直に伝える。
- 問い合わせの主導線は電話。オンライン面談やZoomを前提にしない。

## 言葉のルール

### 基本の言い方

- 「一貫して対応します」「同じ窓口で相談できます」を、事実に沿って使う。
- 中古車では「仕入れ後の状態確認、必要な修理・整備、納車後の点検まで対応」と具体的に言う。
- 修理では「地域密着の整備工場」「板金塗装」「保険修理」を使う。
- CTAは「電話で問い合わせる」「中古車について電話で相談する」のように、行動を具体的に書く。
- 記事の要点は「結論！」など、人が読んで自然な見出しにする。

### 使わない言い方

- 「町の整備工場」「町工場」「少人数の地域工場」
- 「見られます」「整備まで見て」のように、対象や内容が曖昧な表現
- 「絶対安い」「必ずお得」など、根拠を示せない断定
- 「短い結論：」など、制作意図がそのまま見える見出し
- Zoom、オンライン面談を前提とした採用・問い合わせ表現

## 見出しと改行

- 見出しは、まず意味が自然に通る一文にする。見栄えのためだけに文を分断しない。
- 手動改行が必要なときは、句読点・意味の切れ目でのみ行う。`<br class="copy-break ...">` を使い、画面幅ごとに必要な箇所だけ指定する。
- すべての見出しに一律の折返しルールを適用しない。長い見出しは個別にPC・タブレット・スマホを確認する。
- 本文は自然な折返しを基本とし、無理な改行や不自然な短文の連続を避ける。

## デザインと構成

- 基調は白、淡い水色、青系CTA、余白のある清潔感。ロゴの赤は主張させすぎない。
- 背景は青白を交互に使う。ただし、意味が近い小さなセクションはひと続きにしてよい。
- 主CTAは青から水色のグラデーション、副CTAは白地のアウトラインなどで役割を分ける。
- カードでは、見出し、本文、補足、CTAの高さと余白をそろえる。カードを多用しすぎない。
- ヒーローは情報を詰め込みすぎず、最初の画面で主役と相談先が分かるサイズにする。
- 実写真を使うページでは、読みやすさのために淡いオーバーレイを使ってよい。ただし、写真やマスクの形が消えるほど白くしない。
- ヘッダーは企業サイト用と採用用の2種類のみ。フッターは両方で共通化し、車の相談・会社情報・採用情報を整理して載せる。

## SEO / LLO / GEO と人間らしさ

- 検索やLLMのためだけの文章にせず、質問に対して自然な結論、理由、相談できることの順で書く。
- 地域名は必要なページで自然に使う。地域名の過剰な反復は避ける。
- サービスページを受け皿にし、コラムから関連するサービスページへつなぐ。
- 記事の更新日、構造化データ、`llms.txt` は保つが、サイト上の文章は人が読んで違和感がないことを優先する。

## 公開前の確認

1. PCとスマホの両方で、見出し・CTA・カードの高さが崩れていないか。
2. 電話導線が電話番号モーダルまたはスマホの電話発信として自然か。
3. ヘッダーとフッターが共通の最新状態か。
4. 追加した言葉が、このガイドと `implementation-decisions.md` の判断に反していないか。
5. `powershell -NoProfile -ExecutionPolicy Bypass -File .\scripts\verify-site.ps1` を実行する。
