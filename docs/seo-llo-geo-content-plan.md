# SEO / LLO / GEO Content Plan

山本マイカーセンターは、板金塗装・保険修理を主軸に、車検・整備・中古車相談・採用へ展開する。
記事は「質問 -> 短い結論 -> 理由 -> 山本マイカーセンターで相談できること」の順で作る。

## メインサイト優先記事

### 板金塗装・保険修理

- 車のキズ・へこみ修理はどこに頼む？
- 保険修理でお客様がやること・工場が手伝えること
- 車の修理中に代車は借りられる？
- キズ・へこみ修理の費用が車によって変わる理由
- 修理後の納車・引き取りまで相談できる？
- バンパーの擦りキズは放置しても大丈夫？
- ドアのへこみは板金で直せる？交換になる？
- 事故後に車を動かしてよいか迷った時の判断
- 保険を使う修理と自費修理、どちらがよい？
- ディーラー修理と地域密着の整備工場の違い
- 色合わせが難しい車の板金塗装で確認したいこと
- 中古部品を使った修理は選択肢になる？
- 小さなキズでも早めに相談した方がよい理由
- 修理見積りで確認したい項目
- 保険会社指定工場以外に修理を相談できる？

### 車検・整備

- 車検が安い店と整備まで見る店の違い
- 車検前に交換をすすめられやすい部品
- ブレーキの異音はいつ相談するべき？
- オイル交換は車種や年式で何が違う？
- タイヤ交換時に確認したい溝・ひび・年数
- バッテリー上がりを防ぐための点検
- 車検と12か月点検の違い
- 警告灯が点いた時にまず確認すること

### 中古車・新車相談

- 中古車は販売店だけでなく整備工場に相談する選択肢もある
- 中古車購入前に修復歴以外で見るポイント
- 予算内で中古車を探す時に優先順位を決める方法
- 掲載中の在庫がなくても中古車を相談できる理由
- 新車はディーラーで買い、整備は地域密着の整備工場に相談する使い分け
- 社外ナビやドラレコを付けたい時の新車相談

## 採用向け優先記事

- 整備士求人は年収だけで選んでいい？
- 整備士求人の職場見学で聞いておきたいこと
- 未経験から自動車整備士を目指せる？
- 板金塗装の仕事は未経験から目指せる？
- 大手整備工場と地域密着の整備工場、働き方の違い
- 整備士として無理なく働く職場選び
- 整備士を辞めたいと思った時に見る職場の違い

- 日曜・祝日休みの整備士求人を探す時のポイント

- 未経験歓迎の整備士求人で見るべき教育体制

- 地域密着の整備工場で働くメリット

- 応募前に見学だけでもよい理由
