# プロジェクト概要

## サイトの目的

山本マイカーセンター株式会社の公式サイトです。

当初は採用目的が強かったものの、現在は次の3つを同時に伸ばす方針です。

- 中古車相談の問い合わせを増やす
- 板金塗装・保険修理の相談を増やす
- 整備士・板金塗装スタッフの採用につなげる

広告費を大きくかける前提ではなく、SEO、LLO、GEO、Googleビジネスプロフィール、Search Console、GA4、Clarityを使って地道に改善する方針です。

## 事業上の強み

### 板金塗装・保険修理

- 板金塗装に力を入れている。
- 塗装設備も新しいものを入れており、古い印象の修理工場ではない。
- 保険修理の相談が強い。
- 代車あり。比較的きれいな代車を用意しているが、タイミングによって全て出ている場合がある。
- 保険会社とのやり取りは、代行できる部分はサポートする。
- 修理時に車を預かり、代車と入れ替える対応や、修理後のお届けも相談できる。

### 中古車

- Goo-net掲載在庫だけでなく、業者専用オークション等から条件に合う車を探せる。
- 自社で整備・板金塗装まで対応できるため、仕入れ後に必要な整備や外装修理まで同じ窓口で相談できる。
- 修復歴のある中古車も、骨格部位や修理内容を説明し、必要な修理・整備や板金塗装を丁寧に行ってから販売する方針。
- 仕入れ時の写真や修理前写真を出せる場合があり、安い理由を説明しやすい。

### 新車

- スズキの看板があるが、国内全メーカーの新車相談が可能。
- ただし新車販売は年に数台でよく、サイト上では中古車・修理より優先度は低い。
- 新車はディーラーの方が安い場合もあるため、本文では「ディーラー購入も肯定しつつ、購入後の整備・保険・用品取付は地域密着の整備工場に相談できる」という使い分けを推奨する。

### 採用

- 求人媒体では年収だけで比較されやすく、不利になりやすい。
- 強みは、残業や休日、働き方、資格取得支援、未経験から挑戦できる点。
- 大手の整備工場と比較して、働き方や職場の見えやすさを訴求する。
- 「面接」よりも「カジュアル面談」「職場見学」の方が心理的ハードルが低い。

## これまで実施した主な改善

- トップページの訴求整理。
- 採用ページのフォーム確認とGAS改善案作成。
- ヘッダーをメインサイト用と採用用の2系統に整理。
- SEO/LLO/GEO向けに主要サービスページを追加。
- 顧客向けコラムと採用向けコラムを分離。
- 中古車ページを強化し、Goo-net在庫をNetlify Function経由で取得。
- Goo-net在庫カードに画像、年式、走行距離、支払総額、車両本体価格、諸費用を表示。
- 掲載在庫よりも「条件から探す中古車相談」を強く訴求。
- 電話リンクのPC向け挙動を、電話番号表示モーダルに変更。
- GA4、Microsoft Clarity、Search Console向けの土台を追加。
- 自社ブラウザをGA4/Clarity計測から除外するページを追加。
- `llms.txt` と `llms-full.txt` を追加し、LLMがサイト内容を把握しやすい補助ファイルを用意。
